# Подача апелляции — статическая страница

Короткое описание

- Простая статическая вёрстка страницы для подачи апелляции (HTML + CSS + изображения).

Структура проекта

- `index.html` — основная страница.
- `index.css` — стили проекта.
- `vendor/normalize.css` — нормализация стилей.
- `images/` — ресурсы (svg/png и т. п.).
